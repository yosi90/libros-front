# Roadmaps

Indice de verticales documentadas.

- `auth/`: autenticacion, registro y validaciones asociadas.
- `home/`: pantalla de inicio y estado operativo de la API.
- `api-contract/`: alineacion transversal del front con el contrato documentado de la API.
- `notifications/`: sistema global de toasts y compatibilidad con avisos legacy.
- `universes/`: coleccion de universos, libros, sagas y antologias.
- `common/`: componentes compartidos de shell, navegacion y superficies comunes.

## Roadmap activo

- `common/ROADMAP_ACTIVO_redisenio-visual-biblioteca.md`: redisenio visual desktop y alineacion progresiva del shell/perfil/autenticacion con el contrato multiusuario de la API.
